# Multi-tenant SaaS Admin

**Version:** 0.1.0 | **Stack:** React 19 + TypeScript 6 + Vite 8

Dashboard administrativo React SPA que consume la API multi-tenant de [`multi-tenant-saas-api`](https://github.com/anomalyco/multi-tenant-saas-api). Demo/portfolio project.

```bash
npm run dev
```

Open `http://localhost:15001`.
